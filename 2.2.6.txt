На сервере работает скрипт, который раз в сутки загружает файл с логом за прошедшие сутки 
(например, 2015-01-01.log) в HDFS. Работает он следующим образом:

    Локальный файл с логом (2015-01-01.log) копируется в HDFS в папку /tmp/logs/
    Создается папка для нужной даты в /data/logs/ (например, /data/logs/2015-01-01/)
    Из временной папки /tmp/logs/ нужный файл с логом перемещается в папку /data/logs/{date}
    Удаляется папка с логом месячной давности (например, /data/logs/2014-12-01/)

Напишите через запятую последовательность команд (если нужно с опциями) shell-клиента 
для выполнения описанных действий (например: -ls, -mkdir, -du -h, -tail)

Ответ : -put, -mkdir, -mv, rm -r
